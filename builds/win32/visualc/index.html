FreeType 2 Project Files for Visual C++

This directory contains a project file for Visual C++, named freetype.dsp. It will compile the following libraries from the FreeType 2.0.9 sources:

        freetype209.lib     - release build; single threaded
    freetype209_D.lib   - debug build;   single threaded
    freetype209MT.lib   - release build; multi-threaded
    freetype209MT_D.lib - debug build;   multi-threaded

Build directories are placed in the top-level "objs" directory

Enjoy :-)
